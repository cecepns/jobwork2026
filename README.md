# İşwork2025 - Türkçe Landing Page

Türkiye için online iş platformu tanıtım sayfası. React, Vite, TailwindCSS ve AOS animate ile geliştirildi.

## Teknolojiler

- **React** - UI kütüphanesi
- **Vite** - Build tool
- **TailwindCSS** - Utility-first CSS framework
- **AOS** - Scroll animasyonları
- **Para birimi:** ₺ (Türk Lirası)

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

Tarayıcıda `http://localhost:5173` adresini açın.

## Build

```bash
npm run build
```

## Bölümler

- **Hero** - Ana başlık ve CTA
- **Neden Bizi Seçmelisiniz?** - Özellik kartları
- **Gelir Hesaplayıcı** - Günlük/aylık kazanç tahmini
- **İstatistikler** - Platform metrikleri
- **Nasıl Çalışır?** - Adım adım süreç
- **Referanslar** - Çalışan yorumları
- **SSS** - Sıkça sorulan sorular
- **CTA Banner** - Son çağrı
- **Footer** - Alt bilgi ve linkler
